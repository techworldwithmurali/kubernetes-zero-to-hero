
| **Command/Concept**                          | **Description**                                                                                   | **Example**                                              |
|---------------------------------------------|---------------------------------------------------------------------------------------------------|----------------------------------------------------------|
| **`mvn clean`**                             | Cleans the `target` directory, removing all files generated by the previous build.                 | `mvn clean`                                              |
| **`mvn compile`**                           | Compiles the source code of the project.                                                           | `mvn compile`                                            |
| **`mvn test`**                              | Runs the tests in the project.                                                                     | `mvn test`                                               |
| **`mvn package`**                           | Packages the compiled code into a `.jar`, `.war`, or `.ear` file depending on the packaging type. | `mvn package`                                            |
| **`mvn install`**                           | Installs the package to the local repository (`~/.m2/repository`).                                 | `mvn install`                                            |
| **`mvn deploy`**                            | Deploys the project to a remote repository (e.g., Nexus, Artifactory).                             | `mvn deploy`                                             |
| **`mvn validate`**                          | Validates the project to ensure it is correct and all required information is available.          | `mvn validate`                                           |
| **`mvn site`**                              | Generates a site documentation for the project (reports, summaries, etc.).                         | `mvn site`                                               |
| **`mvn dependency:tree`**                   | Displays the project's dependency tree.                                                             | `mvn dependency:tree`                                    |
| **`mvn dependency:copy-dependencies`**      | Copies the dependencies into a specified directory.                                                | `mvn dependency:copy-dependencies`                       |
| **`mvn clean install`**                     | A commonly used combination of commands that cleans, compiles, tests, and installs the project.    | `mvn clean install`                                      |
| **`mvn clean package`**                     | Cleans the project and then packages the project into the desired artifact.                        | `mvn clean package`                                      |
| **`mvn archetype:generate`**                | Generates a new Maven project from an archetype (template).                                        | `mvn archetype:generate`                                 |
| **`mvn help:effective-pom`**                 | Displays the effective POM of the project, which combines all the configurations.                  | `mvn help:effective-pom`                                 |
| **`mvn versions:display-dependency-updates`**| Displays any available updates for project dependencies.                                            | `mvn versions:display-dependency-updates`                |
| **`mvn versions:use-latest-versions`**       | Updates the project's dependencies to their latest versions.                                       | `mvn versions:use-latest-versions`                       |
| **Profiles**                                | Allows different configurations or behaviors for different environments (e.g., dev, prod).         | `<profile><id>dev</id><properties><property>value</property></properties></profile>` |
| **Plugins**                                 | Extend Maven's functionality (e.g., compiler plugin, surefire plugin for running tests).           | `mvn surefire:test`                                      |
| **`<dependency>`**                          | Defines a dependency in the `pom.xml` file.                                                        | `<dependency><groupId>org.apache.commons</groupId><artifactId>commons-lang3</artifactId><version>3.12.0</version></dependency>` |
| **`<build>`**                               | Defines build-related configuration in the `pom.xml`.                                              | `<build><plugins><plugin><groupId>org.apache.maven.plugins</groupId><artifactId>maven-compiler-plugin</artifactId><version>3.8.1</version></plugin></plugins></build>` |
| **`<repositories>`**                        | Specifies remote repositories for dependencies or plugins.                                         | `<repositories><repository><id>central</id><url>https://repo.maven.apache.org/maven2/</url></repository></repositories>` |
| **`<plugins>`**                             | Specifies the plugins used in the project.                                                         | `<plugins><plugin><groupId>org.apache.maven.plugins</groupId><artifactId>maven-jar-plugin</artifactId><version>3.1.0</version></plugin></plugins>` |
| **`mvn clean verify`**                      | Cleans and verifies the project. It runs tests and checks if the project is valid.                 | `mvn clean verify`                                        |
| **`mvn install:install-file`**               | Installs a specific artifact to the local repository.                                              | `mvn install:install-file -Dfile=path/to/artifact.jar -DgroupId=com.example -DartifactId=example -Dversion=1.0 -Dpackaging=jar` |
| **`mvn help:describe`**                      | Provides detailed information about a Maven plugin or goal.                                         | `mvn help:describe -Dplugin=deploy`                       |
| **`mvn dependency:purge-local-repository`**  | Clears the local repository of specific or all dependencies.                                      | `mvn dependency:purge-local-repository`                  |
| **`mvn dependency:analyze`**                 | Analyzes the project's dependencies to identify unused or undeclared dependencies.                 | `mvn dependency:analyze`                                  |
| **`mvn exec:java`**                         | Runs a Java program from the project's classpath.                                                  | `mvn exec:java -Dexec.mainClass=com.example.Main`          |
| **`mvn clean deploy`**                      | Cleans and then deploys the project to a remote repository, such as Nexus.                         | `mvn clean deploy`                                        |
| **`mvn help:effective-settings`**            | Displays the effective settings from all Maven settings files, including user settings.            | `mvn help:effective-settings`                             |
| **`mvn dependency:tree -Dverbose`**          | Displays a verbose dependency tree, showing detailed information about the dependencies.          | `mvn dependency:tree -Dverbose`                           |
| **`mvn compile -DskipTests`**                | Compiles the project but skips the tests.                                                          | `mvn compile -DskipTests`                                |
| **`mvn dependency:resolve`**                 | Resolves the dependencies for the project and downloads them from the remote repository.           | `mvn dependency:resolve`                                  |
| **`mvn site:deploy`**                        | Deploys the generated site to a remote server.                                                     | `mvn site:deploy`                                         |
| **`mvn dependency:build-classpath`**         | Builds the classpath for a project, useful for debugging dependencies.                             | `mvn dependency:build-classpath`                          |
| **`mvn clean validate install -DskipTests`** | Cleans, validates, and installs the project but skips tests to speed up the build.                  | `mvn clean validate install -DskipTests`                  |
| **`mvn versions:revert`**                    | Reverts a version update performed by `mvn versions:use-latest-versions`.                           | `mvn versions:revert`                                     |
| **`mvn clean package -Pprod`**               | Cleans and packages the project using the specified profile (`prod` in this case).                 | `mvn clean package -Pprod`                                |
| **`mvn exec:exec`**                          | Executes a system command in the current project.                                                  | `mvn exec:exec -Dexec.executable=python -Dexec.args=script.py` |
| **`mvn clean install -DskipTests=true`**     | Skips running tests during the clean and install phases.                                           | `mvn clean install -DskipTests=true`                      |
| **`mvn clean test -Dtest=TestClass`**        | Cleans and runs only the specified test class.                                                     | `mvn clean test -Dtest=TestClass`                         |
| **`mvn site:run`**                           | Starts a local server to view the generated Maven site.                                            | `mvn site:run`                                            |
| **`mvn help:effective-plugins`**             | Displays the effective configuration for all plugins.                                               | `mvn help:effective-plugins`                              |
| **`mvn clean install -Dmaven.test.failure.ignore=true`** | Ignores test failures and continues with the build process.                                        | `mvn clean install -Dmaven.test.failure.ignore=true`      |
